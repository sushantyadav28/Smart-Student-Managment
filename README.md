# 🎓 Student Academic Management Dashboard

## 📌 Project Overview

This project is a simple web-based application built using React that helps students manage their daily academic activities. It provides a centralized platform where users can track assignments, monitor attendance, and plan study tasks efficiently.

The main goal of this project is to improve student productivity and organization using a clean and user-friendly interface.

---

## 🚀 Features

* 📋 **Assignments Management**

  * Add, edit, delete assignments
  * Mark tasks as completed
  * Data saved using localStorage

* 📊 **Attendance Tracker**

  * Mark present/absent
  * Automatic attendance percentage calculation
  * Persistent data storage

* 📝 **Study Planner**

  * Add daily study tasks
  * Track completion
  * Helps in daily planning

* 📈 **Dashboard**

  * Displays assignment count
  * Shows attendance percentage
  * Includes charts for visualization

* 🌙 **Dark Mode**

  * Toggle between light and dark themes
  * Improves user experience

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **Routing:** React Router
* **Charts:** Recharts
* **Storage:** LocalStorage

## ⚙️ How to Run the Project

1. Clone the repository

```
git clone <your-repo-link>
```

2. Navigate to project folder

```
cd student-dashboard
```

3. Install dependencies

```
npm install
```

4. Run the project

```
npm run dev
```
